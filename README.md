# Network Port Scanner

## Description
A simple Python-based Network Port Scanner that checks open TCP ports on a target host.

## Features
- Scans ports 1-100
- Uses Python socket programming
- Detects open ports
- Easy to use

## Requirements
- Python 3.x

## How to Run

```bash
python port_scanner.py
```

Enter the target IP address when prompted.

## Example

```text
Enter IP Address: 10.87.213.20

Scanning 10.87.213.20...

Port 80 is OPEN

Scan Complete!
```

## Technologies Used
- Python
- Socket Programming
- VS Code
